This package contains abstract types that deal with menuing within a GUI. The three types available are:

  • XUIMenuBar
  • XUIMenu
  • XUIMenuItem

XUIMenuBars contain XUIMenus which in turn contain XUIMenuItems. However, a XUIMenu can also contain other XUIMenus inside of it. And XUIMenus can also be 'popup' menus which means that they are not anchored to the XUIMenuBar.

For more information about the XUI API, look at the XUI API guide which is part of this download or if you didn't download it, visit The Purnama Project home page. This guide will give you a much deeper understanding as to what can be done with this API. As well, be sure to look at the examples that accompany this API download as well.
